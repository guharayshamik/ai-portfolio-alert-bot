# AI Portfolio Alert Bot

AI-powered portfolio monitoring and market intelligence bot with:

- Telegram alerts
- Macro risk dashboard
- Treasury yield monitoring
- Yield curve monitoring
- VIX monitoring
- Earnings tracking
- Portfolio risk scoring
- Sector weakness detection
- Hourly portfolio ranking

---

## Features

### Macro Dashboard

Monitors:

- VIXY (volatility)
- US 2Y Treasury Yield
- US 10Y Treasury Yield
- Yield Curve (10Y - 2Y)
- DXY Dollar Index
- WTI Crude Oil
- HYG High Yield Bonds
- JNK Junk Bonds

Produces:

- Macro Risk Score (0-100)
- Risk Drivers
- Portfolio Interpretation
